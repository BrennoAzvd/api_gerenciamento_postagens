# Resumo

Esta API faz o gerencimento de postagens. Contendo assim:

- CRUD de usuários;
    
- CRUD de postagens;
    
- Autenticação através de um token JWT
    

## Tecnologias utilizadas:

- Java 17
    
- Spring Boot 3 (Spring 6)
    
- Maven
    
- JPA + Hibernate
    
- PostgreSQL
    
- Docker Compose
    
- JWT
    
- Flyway
